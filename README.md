# 2D-Platformer-Game
 Bracket

A 2D game developed using the Godot Engine. The game includes different scenes, game objects, player mechanics, and assets created/imported for the gameplay experience.

 About The Game

Bracket is a 2D game where the player interacts with the game world, completes objectives, and experiences different gameplay mechanics.

 Features

- 2D gameplay
- Player movement and controls
- Multiple scenes
- Custom game assets
- Game logic implemented using GDScript

 Built With

- Godot Engine
- GDScript
- 2D Game Assets


 How To Run

1. Install Godot Engine.
2. Clone this repository.
3. Open `project.godot` in Godot.
4. Run the main scene.
